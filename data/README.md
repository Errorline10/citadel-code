This folder is for structured data used by the implementation.

The codebase should consume structured project data rather than absorbing full lore prose.

Possible future files:

- `rooms.json`
- `doors.json`
- `objects.json`
- `abilities.json`
- `gates.json`
- `vignettes-index.json`
- `canon-flags.json`

Data in this folder should be treated as implementation data or derived project data, not as the full source of lore canon.
