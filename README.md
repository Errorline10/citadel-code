# The Great Citadel of Knowledge — Code

Implementation repository for **The Great Citadel of Knowledge**.

This repository is the code layer for the Citadel project. It implements structured room, object, ability, gate, map, journal, and interface systems. It does **not** define lore canon.

## Project Boundaries

- Lore canon lives in the Citadel Lore context and related Google Drive documents.
- Code architecture, schemas, implementation notes, build process, and release work live here.
- If implementation reveals a needed lore change, record it as a proposed lore dependency, not accepted canon.

## Current Status

- Repository initialized.
- Architecture not yet selected.
- First implementation target not yet selected.
- Initial scaffold only; no production application code yet.

## Planned Areas

- `docs/` — architecture and implementation notes
